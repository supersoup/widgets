# widgets

## 用 bower 安装第三方库

```
bower install ejs jquery#1.12.4 requirejs requirejs-text require-css mockjs
```

## 已完成

* datePicker: 日历选择器
* fixedHeadTable: 横纵表头固定的表格
* pager: 分页器
* tooltip: 提示框
* numberInput：输入框的数字输入器
* modal: 模态框
* dialog: 气泡对话框
* message: 消息
* notify: 通知
* searchInput: 模糊搜索框
* select：下拉框
* checkboxList: 多选框列表
* transfer: 穿梭选择器

## 计划中

* tree: 树形菜单
* slider: 拖动滑块
* searchSelect: 模糊搜索选择器

## 说明

* 所有组件遵守 AMD 规范
